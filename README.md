## 0x01 介绍

**定位**：搜索引擎+爬虫+GUI，点点点测绘目标收集

**语言**：python3

**功能**：

1. fofa(zoomeye，quake待更新)资产导出
2. Google+Wechat的爬虫
3. 杂七杂八(慢慢更)

## 0x02环境

1. Python环境建议3.7以上，因为有异步
2. 配置文件在config/conf.ini（后续会添加zoomeye，quake等一些api）
3. 入口文件main.py
4. requirements.txt放着需要安装的环境

> pip install -r requirements.txt
